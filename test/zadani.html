Základy JS - opakování

1: Co je JavaScript a k jakým účelům se používá?

Doplň vhodné odpovědi a využij k tomu nečíslovaného seznamu.

Použij v textu aspoň 3 internetové odkazy na zajímavé příklady využití JS!

  • JavaScript je programovací jazyk používaný pro interaktivitu na webu.
  • Umožňuje manipulaci s HTML a CSS.
  • Jeho využití zahrnuje dynamické webové aplikace, hry a server-side aplikace pomocí Node.js.
  • Více o využití: Základy JavaScriptu, MDN web docs, JavaScript.com

2: Jaká pravidla platí pro psaní kódu v JavaScriptu?

Napiš aspoň pět zásad ve podobě číslovaného seznamu.

Použij angličtinu!

  1. Use meaningful variable names.
  2. Avoid using global variables.
  3. Follow consistent indentation and formatting.
  4. Use strict equality (`===`) instead of loose equality (`==`).
  5. Declare variables using `let` or `const` to avoid hoisting issues.

4: Jak je možné v JavaScriptu deklarovat proměnné?

Uveďte na příkladech různé datové typy, které je možné využívat v JS.

Doplňte chybějící kód/text podle vzoru úvodních dvou řádků!

let x = 0; // s přiřazením hodnoty
let x; // bez přiřazení hodnoty
let x = null; // explicitně prázdná hodnota
let y = 42; // celé číslo (typ number)
let z = 3.14; // reálné číslo (typ number)
let e = 1e3; // zápis desetinného čísla pomocí exponentu
let str = "Hello"; // typ string
let prestupny = false // boolean

5: Jaké operátory můžeme používat v JavaScriptu?

Vypište operátory, které patří do uvedených skupin.

U méně obvyklých operátorů dopište do závorky jejich význam!

Matematické

+, -, *, /, % (modulo - zbytek po celočíselném dělení), ** (umocnění)

Přiřazovací

=, +=, -=, *=, /=, %= (přiřadí výsledek operace)

Logické

&& (a), || (nebo), ! (negace)

Porovnávací

==, ===, !=, !==, >, <, >=, <=

Bitové

&, |, ^, ~, <<, >>, >>> (bitové operace s čísly)

Typové

typeof (typ proměnné), instanceof (...)

6: Která popup okna lze využít v JavaScriptu?

K uvedeným příkladům připište do poznámky, co se stane na webové stránce.

Na konci doplňte aspoň dvě varování, proč bychom se měli vyhýbat použití popup oken v reálných webových aplikacích.

alert()

/* ... */
alert('Nezadal jsi správné heslo!');

prompt()

/* Do proměnné password vloží heslo zadané uživatelem */
let password = prompt('Zadejte heslo:');

confirm()

/* ... */
if (confirm('Chceš přejít na stránky Seznam.cz?')) location.href='https://seznam.cz';

Varování: Popup okna mohou narušit uživatelskou zkušenost. Zvyšují riziko zneužití při phishingových útocích.
